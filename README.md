### Basic setup
Clone the project's repository
```bash
git clone https://github.com/IT3R4/envoy-web-app.git
```

Go to the project's directory
```bash
cd envoy-web-app
```

Install dependencies
```bash
yarn
```

Run the project
```bash
yarn start
```

Compile the project
```bash
yarn build
```
